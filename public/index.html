📋 Demo Flow

  1. Call Answer: Your phone will ring, answer it
  2. OTP Prompt: Enter your 4-digit PIN: 1503
  3. Wrong OTP (Optional): Try entering wrong PIN first to see retry
  4. Language Menu: Press 1 for English or 2 for Spanish
  5. Action Menu: Press 1 to hear audio message or 2 to connect with associate
  6. Complete: Call will end after selection

⚙️ Setup Instructions

Prerequisites:

  • Plivo account with Auth ID and Auth Token
  • A verified phone number in Plivo (for caller ID)
  • ngrok installed for local development

Steps:

  1. Create .env file from .env.example
  2. Add your Plivo credentials (PLIVO_AUTH_ID, PLIVO_AUTH_TOKEN, PLIVO_PHONE_NUMBER)
  3. Run ngrok: ngrok http 3000
  4. Update CALLBACK_BASE_URL in .env with ngrok HTTPS URL
  5. Install dependencies: npm install
  6. Start server: npm start
  7. Open http://localhost:3000 in browser
